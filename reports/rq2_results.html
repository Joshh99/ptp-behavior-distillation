Key Findings

  • Airline — L1 vs L0F: L1 PTool scores 77.0% vs L0F CoT 48.3% on exact accuracy (+28.7pp improvement).
  • Tax — L1 vs L0F: L1 PTool scores 99.7% vs L0F CoT 35.3% on exact accuracy (+64.3pp improvement).
  • NBA — L1 vs L0F: L1 PTool scores 44.5% vs L0F CoT 49.5% on F1 macro (-5.1pp decline).
  • NBA degenerate classifier: Despite 80.1% accuracy, L1 achieves only 44.5% F1 macro — well below L0F CoT (49.5% F1). This gap indicates a near-degenerate classifier that over-predicts one class (TN ≈ 0), exposing that accuracy alone is misleading on class-imbalanced NBA verdicts.
  • Airline — Cost: L1 costs $0.00071/problem vs L0F CoT $0.00540/problem — L0F is 7.6x more expensive per query while achieving lower accuracy.
  • Tax — Cost: L1 costs $0.00427/problem vs L0F CoT $0.00499/problem — L0F is 1.2x more expensive per query while achieving lower accuracy.

Methodology

Experimental Setup

  • Model: deepseek-ai/DeepSeek-V3 via Together.ai
  • Temperature: 0  |  Seed: 42
  • Dataset: RuleArena benchmark (Zhou et al., ACL 2025)
    • Airline Baggage Fees — 300 problems
    • Tax Regulations — 300 problems
    • NBA Transactions — 216 problems (boolean verdict, class-imbalanced)

Levels

  • L0 — Python oracle: Deterministic reference calculator; zero LLM cost. Establishes the ceiling.
  • L0F — CoT (free-form): Single LLM call with chain-of-thought prompting. RuleArena-style baseline.
  • L1 — PTool extraction: LLM extracts typed parameters into a structured schema; deterministic calculator applies the rules. Separates language understanding from arithmetic.

Metrics

  • Airline / Tax: Exact accuracy (prediction == ground truth). Tolerance accuracy shown where floating-point rounding causes a gap.
  • NBA: F1 macro (primary) because the verdict labels are class-imbalanced. Exact accuracy shown for reference but is misleading on its own.
