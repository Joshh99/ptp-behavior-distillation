# Experiments

Research experiments organized by phase and research question.

## Structure

### 01-ptp-replication
Baseline implementation and validation of PTP core concepts.

### 02-workflow-spectrum  
Exploration of L0-L3 complexity levels and reliability/generality tradeoffs.

### 03-distillation-methods
Behavior distillation from agentic traces to fixed workflows.

### 04-comparative-analysis
Systematic evaluation and benchmarking across methods and tasks.

## Running Experiments

Each folder contains:
- Implementation code
- Task-specific READMEs with objectives
- Links to results in `/results` directory

See individual folder READMEs for details.
